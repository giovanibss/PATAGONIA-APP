# Patagônia — Dezembro

Roteiro interativo da viagem em família à Patagônia (06 a 17 de dezembro).
Timeline dia a dia, controle de custos, checklist de pendências e hospedagens.

Tudo é editável direto na tela e fica salvo no `localStorage` do navegador.

## Rodar localmente

```bash
npm install
npm run dev
```

Abre em `http://localhost:5173`.

## Publicar no Vercel

1. Suba este repositório no GitHub.
2. No Vercel, clique em **Add New → Project** e importe o repositório.
3. O Vercel detecta o Vite sozinho. Se pedir configuração manual:
   - Framework Preset: **Vite**
   - Build Command: `npm run build`
   - Output Directory: `dist`
4. Deploy.

## Stack

React 18 · Vite 5 · Tailwind CSS 3 · Lucide Icons
